Политика конфиденциальности

AI Site Analyst помогает анализировать сайты, источники трафика, клики и действия посетителей. Мы собираем только те данные, которые нужны для работы аналитики и отчётов.

Какие данные собираются

После установки кода подключения сервис может фиксировать посещения страниц, клики по важным кнопкам, источники переходов, технические данные браузера и общую статистику действий на сайте.

Секретные значения интеграций не показываются в интерфейсе. Данные используются для аналитики, AI-рекомендаций и понятных отчётов по сайту.

Зачем нужен код подключения

Код подключения нужен, чтобы увидеть реальные действия посетителей: клики по WhatsApp, телефону, email, формам и важным кнопкам. Без него доступна только предварительная проверка открытой страницы.

Удаление данных

Пользователь может запросить удаление данных по своему сайту. Для вопросов напишите через страницу контактов.
