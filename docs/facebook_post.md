# Facebook Post — AI Site Analyst

---

Собрал демо-проект AI Site Analyst 👇

Это сервис для бизнес-сайтов, который помогает понять не просто «сколько людей зашло», а где сайт реально теряет заявки.

Как работает:

✅ сайт подключается через небольшой tracker.js скрипт
✅ система собирает посещения, клики и целевые действия
✅ видит клики по WhatsApp, телефону, email, CTA-кнопкам и формам
✅ подключает данные Google Search Console
✅ показывает поисковые запросы, показы, клики, CTR и позиции
✅ строит понятный dashboard для владельца бизнеса
✅ AI анализирует данные и объясняет, что нужно улучшить
✅ Site Score показывает общую оценку сайта 0–100

Главная идея — сделать аналитику понятной для обычного владельца бизнеса, а не только для разработчика или SEO-специалиста.

Такой инструмент можно использовать как дополнительную услугу для клиентов: сайтов, лендингов, SEO-проектов и локального бизнеса.

Проект сделан на Python / FastAPI / PostgreSQL / Gemini / Google Search Console API.

Могу показать демо, как система собирает данные и показывает, где сайт теряет заявки.
